# 🏗️ Training Hub — Construction Learning App

## הרצה מקומית
```bash
npm install
npm run dev
```
פתח: http://localhost:3000

## פריסה ל-Railway
1. `git init && git add . && git commit -m "init"`
2. צור repo חדש ב-GitHub ודחוף לשם
3. Railway → New Project → Deploy from GitHub Repo
4. בחר את ה-repo → Railway מזהה אוטומטית ובונה

**אין צורך בהגדרות נוספות** — railway.json כבר מוגדר.
